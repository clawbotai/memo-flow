## ✨ Logo 设计

### 方案 A: M + 流水
```
    ╭─╮
   ╭╯ ╰╮
  ╭╯   ╰╮
  │  M  │  ← M 字母 + 三条流水线条
  ╰╮   ╭╯
   ╰╮ ╭╯
    ╰─╯
```

### 方案 B: 对话气泡 + 流水
```
  ┌──────┐
  │ 💭🌊 │  ← 灵感气泡 + 流水
  └──────╯
```

### 方案 C: 抽象 M
```
  ╱╲  ╱╲
 ╱  ╲╱  ╲  ← 流畅的 M 形曲线
╱        ╲
```

### 最终推荐：方案 A
**理由**: 简洁、易识别、易实现

## 🎬 动效设计 2.0

### 页面切换
```
进入：从右向左滑动 + 淡入 (300ms)
离开：从左向右滑动 + 淡出 (300ms)
```

### 卡片展开
```
高度：auto (300ms, ease-out)
透明度：0 → 1 (200ms)
```

### 按钮交互
```
悬停：上浮 2px + 阴影 (150ms)
点击：下沉 1px + 缩放 0.98 (100ms)
```

### 新增动效

#### 1. 流水背景
```css
@keyframes flow {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
```

#### 2. 波纹扩散
```css
@keyframes ripple {
  0% { transform: scale(1); opacity: 1; }
  100% { transform: scale(2); opacity: 0; }
}
```

#### 3. 灵感闪烁
```css
@keyframes inspire {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.7; }
}
```
