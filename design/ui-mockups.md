## 🎭 组件库

### 按钮

```tsx
// 主按钮
<button className="btn-primary">
  开始分析
</button>
// 紫色背景，白色文字，圆角 12px

// 次要按钮
<button className="btn-secondary">
  导出 PDF
</button>
// 透明背景，紫色边框

// 图标按钮
<button className="btn-icon">
  📤
</button>
// 仅图标，悬停显示背景
```

### 输入框

```tsx
<input 
  className="input" 
  placeholder="粘贴链接..." 
/>
// 深色背景，紫色聚焦边框，圆角 8px
```

### 卡片

```tsx
<div className="card">
  <div className="card-header">标题</div>
  <div className="card-body">内容</div>
  <div className="card-footer">操作</div>
</div>
// 深色背景，圆角 12px，轻微阴影
```

### 标签

```tsx
<span className="tag">
  #AI
</span>
// 紫色背景，白色文字，圆角 full
```

### 观点卡片

```tsx
<div className="viewpoint-card">
  <div className="viewpoint-title">
    💡 观点标题
  </div>
  <div className="viewpoint-arguments">
    • 论据 1
    • 论据 2
  </div>
</div>
// 渐变边框，悬停发光效果
```
