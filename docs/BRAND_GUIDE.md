## 🎯 品牌定位

### 品牌名称
**MemoFlow**  
/ˈmɛmoʊ floʊ/

### 品牌定义
AI 驱动的内容分析与创作助手，让灵感和笔记如流水般自然流淌。

### 品牌口号
**英文**: Let Your Ideas Flow  
**中文**: 让灵感自然流淌  
**简短版**: Flow Your Notes

### 品牌价值
- 🌊 **流畅** - 从消费到创作，一气呵成
- 💡 **灵感** - 激发创意，源源不断
- 🧠 **智慧** - AI 赋能，深度分析
- ✍️ **创作** - 高效产出，多平台适配

## 🎨 视觉识别系统

### Logo 设计

#### 主 Logo
```
MemoFlow
├─ 图标：M + 流水线条
├─ 字体：无衬线字体
└─ 排列：图标 + 文字（水平）
```

#### Logo 使用规范
- **最小尺寸**: 24px 高度
- **安全空间**: Logo 高度的 1/4
- **背景**: 纯色或渐变，避免复杂背景

#### Logo 变体
```
1. 完整版：图标 + MemoFlow
2. 图标版：仅图标（favicon/头像）
3. 文字版：仅 MemoFlow 文字
```

### 品牌色

#### 主色
| 名称 | 色值 | 用途 |
|------|------|------|
| Flow Blue | `#2563EB` | 主按钮、链接、强调 |
| Flow Light | `#60A5FA` | 悬停状态、次要元素 |
| Flow Dark | `#1E40AF` | 深色模式、强调 |

#### 辅助色
| 名称 | 色值 | 用途 |
|------|------|------|
| Growth Green | `#10B981` | 成功状态、积极反馈 |
| Warm Purple | `#8B5CF6` | 渐变、特殊强调 |
| Sunset Orange | `#F97316` | 警告、通知 |

#### 中性色
| 名称 | 色值 | 用途 |
|------|------|------|
| Pure White | `#FFFFFF` | 浅色模式背景 |
| Pure Black | `#000000` | 深色模式背景 |
| Gray 100-900 | `#F3F4F6` - `#111827` | 文字、边框、背景 |

#### 渐变方案
```css
/* 主渐变 */
background: linear-gradient(135deg, #2563EB 0%, #60A5FA 100%);

/* 流水渐变 */
background: linear-gradient(135deg, #60A5FA 0%, #2563EB 50%, #1E40AF 100%);

/* 彩虹渐变 */
background: linear-gradient(135deg, #8B5CF6, #2563EB, #10B981);
```

### 字体系统

#### 英文字体
```
Primary: Inter (无衬线)
- 字重：400, 500, 600, 700
- 用途：界面文字、标题

Secondary: Georgia (衬线)
- 字重：400, 700
- 用途：长文本阅读
```

#### 中文字体
```
Primary: Noto Sans SC (思源黑体)
- 字重：400, 500, 600, 700
- 用途：界面文字、标题

Secondary: Noto Serif SC (思源宋体)
- 字重：400, 700
- 用途：长文本阅读
```

#### 字号规范
```

3xl: 30px (1.875rem) - 超大标题

```

## 📝 文案规范

### 语调风格
- **友好** - 像朋友一样对话
- **专业** - 但不高冷
- **简洁** - 不啰嗦
- **鼓励** - 正向反馈

### 文案示例

#### 按钮文案
```
✅ 开始分析
❌ 点击这里进行分析

✅ 生成笔记
❌ 执行笔记生成操作

✅ 导出
❌ 执行导出功能
```

#### 错误提示
```
✅ 链接格式不正确，请检查后重试
❌ 错误：无效输入

✅ 分析失败，请稍后重试
❌ 系统错误 500
```

#### 成功反馈
```
✅ 分析完成！已生成 3 个核心观点
❌ 操作成功

✅ 笔记已保存
❌ 保存完成
```

#### 空状态
```
✅ 还没有笔记，粘贴链接开始分析吧
❌ 无数据

✅ 分析结果会显示在这里
❌ 内容为空
```

### 禁用文案
```
❌ 用户
✅ 你

❌ 系统
✅ MemoFlow

❌ 功能
✅ 特性
```

## 🖼️ 图像规范

### 插图风格
- **简洁** - 线条简洁
- **现代** - 扁平化设计
- **温暖** - 圆角、柔和
- **一致** - 统一风格

### 图标规范
```
尺寸：16px, 24px, 32px, 48px, 64px
风格：线性图标，2px 描边
圆角：2px
颜色：跟随文字颜色
```

### 图片使用
- **质量**: 最小 720p，推荐 1080p
- **格式**: WebP 优先，PNG 备选
- **压缩**: 质量 80%，大小 < 500KB
- **比例**: 16:9（视频封面），3:4（小红书）

## 📱 界面规范

### 间距系统
```
4px (0.25rem)   - 最小间距
8px (0.5rem)    - 小组件间距
12px (0.75rem)  - 组件内间距
16px (1rem)     - 标准间距
24px (1.5rem)   - 大间距
32px (2rem)     - 区块间距
48px (3rem)     - 大区块间距
64px (4rem)     - 页面间距
```

### 圆角规范
```
sm: 4px    - 小按钮、标签
md: 8px    - 卡片、输入框
lg: 12px   - 大卡片、模态框
xl: 16px   - 特殊组件
full: 9999px - 头像、圆形按钮
```

### 阴影规范
```css
/* 轻阴影 */
box-shadow: 0 1px 2px rgba(0, 0, 0, 0.05);

/* 标准阴影 */
box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);

/* 重阴影 */
box-shadow: 0 10px 15px rgba(0, 0, 0, 0.1);

/* 悬浮阴影 */
box-shadow: 0 20px 25px rgba(0, 0, 0, 0.15);
```

### 动效规范
```
持续时间:
- 快速：100ms (微交互)
- 标准：200ms (按钮、卡片)
- 慢速：300ms (页面切换)
- 很慢：500ms (大型动画)

缓动函数:
- 标准：ease-in-out
- 进入：ease-out
- 离开：ease-in
- 弹性：cubic-bezier(0.68, -0.55, 0.265, 1.55)
```
