# 🧠 MemoFlow - 产品命名方案

## 📋 命名原则

1. **易记** - 简短、朗朗上口
2. **相关** - 体现产品功能（内容分析/笔记/创作）
3. **独特** - 避免与现有产品重名
4. **可注册** - 域名/商标可用
5. **国际化** - 中英文都友好

- 简洁
- 适合科技公司

## 📝 命名检查清单

在最终确定前，请检查：

- [ ] 域名是否可注册
- [ ] 商标是否可用
- [ ] 社交媒体账号是否可用
- [ ] 是否有负面含义（多语言检查）
- [ ] 是否容易拼写错误
- [ ] 是否容易发音
- [ ] 是否与竞品区分明显
