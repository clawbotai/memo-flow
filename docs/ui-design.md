# 🎨 MemoAI - UI 设计方案

## 🎯 设计风格

**简洁 · 专业 · 高效**

- 主色调：紫色系 (#8b5cf6)
- 辅助色：渐变彩虹
- 主题：深色优先，支持浅色
- 字体：Inter + Noto Sans SC

## 🎭 组件设计

### 按钮

```tsx
// 主按钮
<button className="btn-primary">开始分析</button>
// 渐变背景，圆角 12px，白色文字

// 次要按钮
<button className="btn-secondary">导出 PDF</button>
// 透明背景，紫色边框

// 图标按钮
<button className="btn-icon">📤</button>
// 仅图标，悬停显示背景
```

### 卡片

```tsx
<div className="card">
  <div className="card-header">标题</div>
  <div className="card-body">内容</div>
  <div className="card-footer">操作</div>
</div>
// 深色背景，圆角 12px，轻微阴影
```

### 输入框

```tsx
<input className="input" placeholder="粘贴链接..." />
// 深色背景，紫色聚焦边框，圆角 8px
```

### 标签

```tsx
<span className="tag">#AI</span>
// 紫色背景，白色文字，圆角 full
```
